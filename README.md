# Termux-Setup
✅Auto update &amp; upgrade </br>
✅ Install essential tools </br>
✅ Python, Ruby, Node, Git, Curl, Wget </br>
✅ Compiler tools (clang, make) </br>
✅ Network tools (nmap, net-tools, openssh) </br>
✅ zip/unzip, vim, nano, htop, etc.</br>


### How to Install
``` Bash
 cd Termux-Setup
 chmod +x termux-setup.sh
 bash termux-setup.sh
```
